# 🔗 URL Shortener

A simple and responsive web application that allows users to shorten long URLs instantly using an external API. Built with **HTML, CSS, and JavaScript**, this project demonstrates API integration, DOM manipulation, and responsive UI design.

---

## 🚀 Features

* 🔗 Shorten long URLs instantly
* 📋 One-click copy to clipboard
* 📱 Responsive design (mobile-friendly)
* 🍔 Hamburger menu for smaller screens
* ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla JS)** – Logic & API handling
* **API Used:** shrtco.de

---

## 📂 Project Structure

```
URL-Shortener/
│── index.html     # Main HTML file
│── style.css      # Styling
│── app.js         # JavaScript logic
│── images/        # Assets (favicon, etc.)
```

---

## ⚙️ How It Works

1. User enters a long URL
2. Form submission is intercepted using JavaScript
3. A request is sent to the shortening API
4. Shortened URL is displayed dynamically
5. User can copy the link with a single click

## 💻 Installation & Setup

Clone the repository:


```bash

```

Simply open `index.html` in your browser.
